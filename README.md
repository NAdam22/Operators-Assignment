# Operators-Assignment

Employee Operator Overloading Console App
Project Description:

This C# console application demonstrates operator overloading by comparing two Employee objects using the == and != operators.
Employees are considered equal only if their Id values match, regardless of their names.

Project Structure:

Employee.cs

Contains the Employee class

Properties: Id, FirstName, LastName

Overloaded == and != operators

Overrides Equals() and GetHashCode()

Program.cs

Instantiates two Employee objects

Compares them using the overloaded operators

Displays the comparison result in the console

Features:

Custom comparison of objects using operator overloading

Proper implementation of == and != operators (overloaded in pairs)

Safe null checking and reference checking

Fully commented code for easy understanding


Example Output :        Are the two employees equal? True
